== Description ==

BlogLogistics Limited Blog Writing Access allows Editors, Authors, and Contributors to create and edit blog posts, while preventing media library access, uploads, and publishing. Subscribers and other non-writing users are redirected away from wp-admin. This plugin changes WordPress role capabilities on activation, specifically for Editors, Authors, and Contributors.

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/.
2. Activate the plugin in WordPress.
3. Test with Administrator, Editor, Author, Contributor, and Subscriber accounts before using as full production.
